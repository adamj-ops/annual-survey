# Vasculearn Network Design System

## Color Palette (from thrombosis.org)

Based on analysis of thrombosis.org website:

### Primary Colors
- **Dark Purple (Primary)**: `#4A148C` or similar - Used for buttons, headers, footer
- **Light Purple (Secondary)**: `#9C27B0` or similar - Used for secondary buttons, accents
- **White**: `#FFFFFF` - Backgrounds, text on dark backgrounds
- **Dark Grey/Black**: `#212121` or similar - Primary text color

### Typography
- **Font Family**: Clean sans-serif (Inter, Open Sans, or similar)
- **Headings**: Bold, larger sizes
- **Body Text**: Regular weight, readable sizes

### Component Styles
- **Buttons**: Rectangular with slight rounding (border-radius: 4-6px), solid fill, white text
- **Input Fields**: White background, subtle border, rectangular shape
- **Layout**: Clear sectioning with alternating backgrounds, ample white space

## Design Tokens

These colors will be integrated into Tailwind CSS configuration to match the thrombosis.org aesthetic.
